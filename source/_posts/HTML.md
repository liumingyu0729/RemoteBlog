超文本标记语言 .html .htm 大小写不敏感

~~~html
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>第一个</title>
</head>
<body>
    <h1>第一个标题</h1>
    <p>第一个段落。</p>
</body>
</html>
~~~

- “<!DOCTYPE html>” 声明HTML5 文档
- “<html>” 根元素
- “<head>” 元素包含了文档的元（meta）数据
- "<title>" 标题
- "<body>" 内容
- "<h1>" 大标题 h1-h6
- "<p>" 段落

标签 

~~~html
<b> </b>
~~~

链接“<a>” 

~~~html
<a href="http://www.baidu.com">这是一个链接</a>
~~~

图像“<img>”

~~~html
<img src="/images/logo.png" width="258" height="39" />
~~~

换行

~~~html
<br>
~~~

关闭空元素

~~~html
<br />
~~~

HTML属性
元素可以设置属性（推荐小写）

注释 “<!-- 这是一个注释 -->”

水平线 “<hr>”

文本格式化

|   标签   |   描述   |
| :------: | :------: |
|   <b>    |   粗体   |
|   <em>   |   着重   |
|   <i>    |   斜体   |
| <small>  |  小号字  |
| <strong> | 加重语气 |
|  <sup>   |   下标   |
|  <sub>   |   上标   |
|  <ins>   |   插入   |
|  <del>   |  删除字  |
